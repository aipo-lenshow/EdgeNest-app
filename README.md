# EdgeNest — Desktop Releases

EdgeNest macOS 客户端的下载与自动更新源。最新安装包见 [Releases](../../releases/latest)。

This repository hosts the EdgeNest macOS app downloads and its auto-update feed (`appcast.xml`). Get the latest installer from [Releases](../../releases/latest).

## 安装 / Install

1. 下载最新的 `EdgeNest-<版本>.dmg` 并打开，把 EdgeNest 拖进「应用程序」。
2. 应用会自动检查更新；也可以在 设置 → 关于 → 检查更新 手动检查。

1. Download the latest `EdgeNest-<version>.dmg`, open it, and drag EdgeNest into Applications.
2. The app checks for updates automatically; you can also check manually from Settings → About → Check for updates.
